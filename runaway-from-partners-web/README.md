# Убеги от партнёров (MVP)

Небольшая 3D хоррор-сцена в браузере на Three.js:
- главное меню с кнопкой `Начать игру`
- игрок от первого лица (`WASD` + мышь)
- комната с кроватью, тумбочкой, столом, лампой и чемоданом
- записка на чемодане: `F` чтобы прочитать
- после чтения чемодан автоматически добавляется в инвентарь и появляется "в руках"

## Управление

- `W`, `A`, `S`, `D` — движение
- Мышь — обзор
- `F` — взаимодействие (чтение/закрытие записки)
- `Esc` — пауза (выход в меню)

## Быстрый запуск

Требуется Node.js 20+ (рекомендуется LTS).

```bash
cd runaway-from-partners-web
npm install
npm run dev
```

Открой в браузере адрес из терминала (обычно `http://localhost:5173`).

## Production-сборка

```bash
cd runaway-from-partners-web
npm run build
npm run preview
```

Готовые статические файлы будут в папке `dist/`.

## По ассетам и звуку

В текущем MVP используются встроенные процедурные материалы и генерация фонового эмбиента через Web Audio (без внешних файлов), чтобы проект запускался сразу.
На следующем шаге можно подключить открытые CC0-ассеты (модели/текстуры/звуки) с явной фиксацией лицензий.
